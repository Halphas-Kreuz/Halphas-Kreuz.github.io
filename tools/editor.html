Local Publisher _ □ ×

这是一个“AO3式”的本地发布器：你写富文本，它会把内容生成纯 HTML 并写入仓库的 /posts/，同时更新首页时间轴/标签/分类页。

注意：为了能直接写入文件，这个页面需要 Chromium 系浏览器（支持 File System Access API）。 如果你的浏览器不支持，会退化为“下载生成的 HTML 文件”。
